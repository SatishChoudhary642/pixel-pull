How it works
01 — Photographer

Upload event photos

Sign in and upload your album. Every batch gets a unique 6-character access code to share with attendees.

02 — Background processing

Faces are extracted

A C++ worker runs HOG detection and ResNet-34 embedding on every face in every photo. Stored as 128-dimensional vectors.

03 — Attendee

Enter code + selfie

Attendees enter the access code and upload a selfie. No account needed. Works on any device.

04 — Match

Get your photos

Euclidean distance comparison across all stored face vectors returns only the photos containing the attendee's face.
